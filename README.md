# MC Skins Gallery

Una galería de skins de Minecraft creada por DarianGMR.

## Características
- Visualización 3D de skins
- Subida de skins personalizadas
- Búsqueda por nombre
- Vista previa antes de subir
- Sistema de reemplazo de skins

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- skinview3d (para visualización 3D)
- Font Awesome (para iconos)
